To test the program, follow these steps:

1. Download the file (Go to the green button and press Download ZIP). 

2. Extract the folder from the ZIP into a desired location.

3. Use the terminal or CMD to access the location of the folder where the program is

4. Write the following: python VirusGrowthSimulation.py

5. It that does not work, try: python3 VirusGrowthSimulation.py

6. If the graphs don't show, you may need to install Matplotlib, a Python library. Then follow the instructions of this link: https://matplotlib.org/users/installing.html
